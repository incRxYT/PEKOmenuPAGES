PEKOmenu is a simple mod menu.

GDH-style hack menu for Geometry Dash 2.2081 with essential features.

Feature Overview
  • Auto Retry
  • Instant Complete
  • No Death Effect
  • Safe Mode
  • Practice Mode
  • Death Counter
  • Speed Hack
  • TPS Unlocker
  • Frame Stepper
  • Startpos Switcher
  • Labels (FPS, Attempts, Progress)
  • Hitboxes / Visual Tweaks

  1. Download the latest version from GitHub Releases
  2. Open Geode → Settings → Install from File
